Time series binning

The bin width of the light curve is controlled by two input parameters timebinstyle and timebinwidth. If timebinstyle is set to intrinsic then the light curve is binned at the frame time of the observation. This may be overridden by setting timebinstyle=parameter in which case the bin width is taken from the timebinwidth parameter, e.g. for a 10 second binning 

   etimeget timebinstyle=parameter timebinwidth=10

For EPIC-pn observations the frame time is set using the observing mode and a look-up table (taken from A&A, 365, L20). The relationship between frame time and observing mode is not constant for the MOS and so the frame time is read from the keyword FRMTIME present in the EXPOSU extensions of the event file.

The start and end times for the source and background light curves may be defined using the timemin and timemax parameters.
