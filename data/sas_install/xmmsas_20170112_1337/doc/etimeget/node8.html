Using the intrinsic frame time as a bin width

etimeget filestem=myhmxb srcexp="((X,Y) IN circle(25572,-24635,1200))"
backexp="((X,Y) IN circle(23000,-28000,1340))"
table=P0129360201PNS020PIEVLI0000.FIT timebinstyle=intrinsic

This command will produce a source and background time series binned at the frame time of the observation, e.g. 0.07336 seconds for Full Frame EPIC-pn.
