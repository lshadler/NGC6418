A light curve with 10 second binning

etimeget filestem=myagn srcexp="((DETX,DETY) IN circle(572,-635,800))"
table=P0129360201M1S004MIEVLI0000.FIT timebinstyle=parameter timebinwidth=10

This command line produces the file myagn_srcrate.lc and myagn_bgdrate.lc
