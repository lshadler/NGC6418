opticsxy: compute optical axis/focal plane intersection point

  • SYNOPSIS

        opticsxy
  • DESCRIPTION
    opticsxy computes for each instrument the intersection point of the optical axis of the respective telescope and the focal plane. The result in given in CHIPCOORD coordinates of the CCD containing the intersection point. The numbers can be used as CCD MiscData attributes OPTICS_X/OPTICS_Y (+OPTICS_CCD).
  • REFERENCE
    See task epicbscalgen and cal documentation for definition of boresight related coordinate frames.
