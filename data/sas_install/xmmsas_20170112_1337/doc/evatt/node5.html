EPN_REJECTION_MASK = OUT_OF_FOV + IN_BAD_FRAME + COSMIC_RAY + ON_BADPIX + SECONDARY + TRAILING + MIP_ASSOCIATED + OUT_OF_CCD_WINDOW Note: OUT_OF_CCD_WINDOW is missing in the current implementation. This is, however, not really relevant as events with this flag cannot be handled in tasks later than epframes. The flag is set and the events are already rejected within the task. The attribute IN_BAD_FRAME is set if OUT_OF_CCD_WINDOW is set and IN_BAD_FRAME is in EPN_REJECTION_MASK.
