Meta Index / Home Page


Description

This task returns the following statistical information for each column in the list specified by the parameter column, present in the table specified by the parameter table:

  • The minimum value
  • The maximum value
  • The mean average
  • The root mean squared
  • The sum
  • The total number of elements considered.

The results are written to the standard output as well as the text file specified by the parameter resultfile.
