Background algorithms

A parameter backgroundmethod determines which background method to be used with aperture photometry. There are 5 methods that can be used, and the default method of one which is the current SAS method is recommended since it is the one that has been used in the calibration. The other methods should be used with care.

  • Method 2- Mean
  • Method 3- Median
  • Method 4- Mode algorithm
