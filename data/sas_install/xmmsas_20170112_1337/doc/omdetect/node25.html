General Comments

Omdetect will find sources on an image using both a peak-finding and connected-pixel algorithm. The former will mainly detect ``point'' sources and the latter ``extended'' ones. The peak-finding algorithm is used first to try to locate all point sources, in order of decreasing brightness. This algorithm mainly overcomes problems with modular-8 patterns around bright sources and scattered light features. When the detection algorithms have finished, photometry is performed on the sources and then the flagging algorithms are run.
