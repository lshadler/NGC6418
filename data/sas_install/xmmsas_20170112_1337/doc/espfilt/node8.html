Customisation

Command line options may be used to modulate certain parameters. These modifications include:

  • eventset parameter, any valid MOS or PN eventlist. can also be a list of event lists. Espfilt will loop through each.
  • The parameter smooth allows the user to select a smoothing factor (with default of 60).
  • The parameter binsize allows the user to select a time bin width (in seconds) for the lightcurves created (default is 60)
  • The parameters specchanmin and specchanmax allow a user to set upper and lower energy ranges, in eV (effectively channel number) paired with the parameter withspecranges
  • The parameter ratio allows the user to select a threshold for the ratio of the counts per pixel in the 8-12keV range between a FOV annulus and the counts in unexposed corners (default is 1.2)
