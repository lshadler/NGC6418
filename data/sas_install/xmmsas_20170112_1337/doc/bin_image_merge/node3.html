Meta Index / Home Page


Description

bin_image_merge produces binned count rate and count-rate uncertainty images for merged observations using the output of the task merge_comp_xmm. For each unmasked and binned pixel, the program will determine the average count rate and the count rate uncertainty. The assumption is that the uncertainty is dominated by the counting statistics and the the systematics of the background modeling.

Warning and requirements: bin_image_merge is part of the package esas, integrated into SAS, but (still) limited to work within the esas data reduction scheme. This is specially true wrt input files structure and names. In particular, bin_image_merge assumes that individual observations have been processed and subsequently mosaicked by the task merge_comp_xmm.
