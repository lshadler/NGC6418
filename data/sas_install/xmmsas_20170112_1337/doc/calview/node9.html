Comments

  • Not all the calibration viewer variables are meaningful to all viewables. At the moment there is no way to know which variable affects which viewable, although it is fairly easy to guess.

  • The calibration view server may fail to initialize if the Calibration Index File does not contain all of the required datasets. At the moment the only way of knowing if all the required datasets are available is to try and run calview.

    The Calibration Index File and the CCF that the users receive with their data is complete and can be used with calview.
