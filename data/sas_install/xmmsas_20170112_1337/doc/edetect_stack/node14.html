Input Files

Default usage:

  1. PPS product by task atthkgen: ATTHK FITS file containing attitude information (per observation)
  2. PPS product e.g. by meta-tasks epchain / emchain or epproc / emproc: event lists (per observation and instrument)
  3. PPS product by task odfingest: ODF summary file (per observation)

Alternative usage, when running certain stages of edetect_stack:

  1. PPS products or intermediate products of former calls to edetect_stack: input files for subsequent task stages (per pointing, instrument, and energy band). E.g. images, exposure maps.
