The SRCLIST table

This table will contain astrometric, photometric, source-parameterisation and source-flagging parameters. The number of columns will depend on the number of different filters that are present from the input source-list files.

The following sections describe the individual columns in the table.


Subsections
  • SRCID
  • RA and DEC
  • RA_CORR and DEC_CORR
  • POSERR
  • LII and BII
  • FILTER_SIGNIFICANCE
  • FILTER_RAW_RATE
  • FILTER_CORR_RATE
  • FILTER_CORR_RATE_ERR
  • FILTER_INS MAG
  • FILTER_INS MAG_ERR
  • FILTERS_INS
  • FILTERS_INS_ERR
  • FILTER_STD(COLOUR)
  • FILTER_STD_ERR(COLOUR)
  • FILTER_AB_FLUX
  • FILTER_AB_MAG
  • FILTER_MAJOR_AXIS
  • MEAN_MAJOR_AXIS
  • FILTER_MINOR_AXIS
  • MEAN_MINOR_AXIS
  • FILTER_POSANG
  • MEAN_POSANG
  • FILTER_QUALITY_FLAG
  • QUALITY_FLAG_COMB
  • FILTER_CONFUSION_FLAG
  • CONFUSION_FLAG_MAX
  • FILTER_EXTENDED_FLAG
  • EXTENDED_FLAG_MAX
