RA and DEC

These columns will contain the Right Ascension and Declination (J2000) of each merged source. If the parameter alignaxes was set to true (default) then the RA and Dec coordinates of each of the individual sources that constitute a merged source may have small corrections added.

All the entries in the table will be in order of increasing RA value.
