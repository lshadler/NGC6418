Catalogue file format

Unfortunately, at the present time no catalogue file is included with an ODF to enable a user to re-compute the astrometry correction. However, CB is happy to create the file for anybody, on request. The catalogue file has to be in FITS file format and contain the following columns in a RAWRES table.:

  • CAT_RA
  • CAT_DEC
  • CAT_RADEC_ERR
  • BMAG
