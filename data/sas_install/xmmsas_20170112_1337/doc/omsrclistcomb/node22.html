FILTER_INS MAG

For each filter there is a column for the source instrumental magnitudes. These values are computed from the averaged corrected count-rates that are tabulated in the CORR_RATE columns values from the individual sources that were merged into a source, using the appropriate calibration constants for the filter that convert count=rates to instrumental magnitudes. These magnitudes can also be referred to as ``VEGA'' magnitudes. No magnitudes are computed if the corrected count-rate exceeds 1000.
