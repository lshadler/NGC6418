Future developments

It is anticipated that more functionality shall be added to this package in the future. The proposed task movecalc is an important part in the analysis of data from moving objects but can only address spatial aspects. Another related task might be regionunmove to convert a spatial X_MOV/Y_MOV region selection into X/Y space.
