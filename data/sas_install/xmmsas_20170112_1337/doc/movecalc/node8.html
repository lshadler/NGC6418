Input Files

  1. event list with sky image columns xcol/ycol generated by attcalc
  2. data set generated by task iauephem defining the position of the moving object as a function of time. Please note that iauephem does not have to be used to prepare the input but any tools which generates a data set with a single binary table (with any name) and the folling minimal column structure:
    column name meaning type units
    MJD modified Julian Day number real64 d
    RA apparent geocentric Right Ascension real32 deg
    DEC apparent geocentric Declination real32 deg
    DELTA distance to center of earth real32 AU
