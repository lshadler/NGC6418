Implicit selection through the mode

If you opt to select the data modes that should be processed, then all the exposures in a given mode are processed. The relevant parameters are:

  • selectmodes
  • imaging, rimaging, timing, ctiming for emproc.
  • imaging, timing, burst for epproc.

If both selectmodes and withinstexpids are given, the former is ignored.

By default all available exposures are processed.
