Timing modes

The processing of timing modes may require that the user specify additional parameters to the tasks epframes and emframes. You should use the parameters withsrccoords, srcra, and srcdec. For PN you can optionally use timingsrcposition in RAWY coordinates. In most cases the default values are adequate.
