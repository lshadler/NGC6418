Comments

  • A new attribute SCFACTOR is introduced into the attribute table of the output file. The value of this attribute corresponds to the automatically computed scaling factor for scaling the calibrated image before its subtraction from the input image. This value can be used for adjusting the scaling factor and obtaining better results by setting the parameter scalingfactor to a value defined by the discretion of the user.
