Future developments

  • It is likely that the calculation of the scaling factor for adjusting the image with the calibrated scatted light feature to the input image would need further improvements. In some cases the scaling factor might be slightly overestimated (as it is seen, e.g., in Fig.5).

  • Currently the calibration files with the central scattered light feature are not included into the calibration (CAL) system of SAS, but the proper way of using these files would be through the standard CAL routines.
