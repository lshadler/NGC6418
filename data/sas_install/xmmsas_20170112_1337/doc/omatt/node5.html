Catalogue file format

The USNO subset catalogue file could be included into the observation ODF to enable users to re-compute the astrometry correction. Then the name of this subset should be indicated to the task omatt in the parameter catfile, while the parameter usecat should be set to `yes'. The USNO subset catalogue file has to be in the FITS file format containing the following columns in the extension named RAWRES:

  • CAT_RA (USNO right scension coordinate)
  • CAT_DEC (USNO declination coordinate )
  • CAT_RADEC_ERR (RMS for the RA and DEC coordinates)
  • BMAG (B-band magnitudes)
  • possibly RMAG (R-band magnitudes)
