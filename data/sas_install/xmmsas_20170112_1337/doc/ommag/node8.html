Output Columns modified

  1. CORR_RATE - The count rate after the following corrections have been applied;

    1) A further point-spread function correction may be applied, to extrapolate the counts from 12 to 35 pixels (presently only done for UV observations) 2) Correction for dead-time. 3) Correction for the OM sensitivity-degradation. A key-word TDS_CORR is added to the table to record the correction value.

  2. CORR_RATE_ERR - The error in the corrected count rates.
