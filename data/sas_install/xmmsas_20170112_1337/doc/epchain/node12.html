Errors

This section documents warnings and errors generated by this task (if any). Note that warnings and errors can also be generated in the SAS infrastructure libraries, in which case they would not be documented here. Refer to the index of all errors and warnings available in the HTML version of the SAS documentation.

odferror 
 SAS_ODF environment variable does not exist and no data directory given
 
indirerror 
 given data directory could not be found
 
odffileserror 
 no valid file found for given specifications (data mode, exposure, etc.)
 
badpixseterror 
 parameter does not contain CCD number place-marker ##
 
refpointerror 
 no values given for RA, DEC in the case of refpointlabel=user
 
attitudeerror 
 no values given for RA, DEC, PA in the case of attitudelabel=fixed
 
ccdlisterror 
 list of CCDs cannot be expanded, e.g. ``2-4-6'' or ``3,-5''
 
ccderror 
 CCD number out of range [1-12]
 
exposureerror 
 exposure not found in ODF directory
 
constituenterror 
 one (or more) of the constituent calls ended in error
 
SAS_CCFwarning 
 ccf.cif or environment variable SAS_CCF not found
corrective action: continue, check SAS_CCF environment variable
 
notInODFwarning 
 requested CCD file seems to be non-existent in ODF, skip processing for this CCD
corrective action: continue, check ODF or give other CCD list parameter when reprocessing
