Optical loading

As the final event file does not contain events below 150 eV, the information to assess the presence of optical loading in a straight forward way is not available in the event file anymore. Parameter optloadingimage therefore controls the optional creation of an image in the ADU 20-30 range where optical loading is easily visible.
