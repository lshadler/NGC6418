Event attributes

epframes creates event quality flags in column FLAG (Int32). It uses the common MOS/pn event related flag code (see [5]) and sets the following event attributes (other flags are set by the task epevents):

FLAG
Attribute Meaning (information)
IN_SPOILED_FRAME wrong order of events
  or events not sorted in time
ON_OFFSET_COLUMN event on column with higher offset
NEXT_TO_OFFSET_COLUMN event on column next to one with higher offset

Attribute Meaning (rejection)
IN_BAD_FRAME event belongs to bad frame
OUT_OF_CCD_WINDOW event outside the CCD readout window
MIP_ASSOCIATED events associated with MIP events
