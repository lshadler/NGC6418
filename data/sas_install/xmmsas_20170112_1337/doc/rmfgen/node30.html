Comments

  • The RGS is addressed by the task rgsrmfgen.

  • The size of the o/p RMF is dependent on the spectrum binsize and the number of bins in the energy grid, and can be very large. Heavy consumption of disk space is expected. Memory consumption will also be high as the full RMF is held in memory before being written to disk. The execution time is particularly large, potentially taking hours, if the pile-up correction option is applied.

  • The task directly uses a section of code from the arfgen task. The debug and warnings from this code may appear to be related to arfgen but they are equally relevant for this task.
