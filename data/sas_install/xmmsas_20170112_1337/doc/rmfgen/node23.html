    • A table labelled MATRIX, containing the following columns

      • A 32-bit real column labelled ENERG_HI
      • A 16-bit integer column labelled N_GRP
      • A 16-bit integer vector column labelled F_CHAN
      • A 16-bit integer vector column labelled N_CHAN
      • A 32-bit real vector column labelled MATRIX

      and the following attributes:

      • TELESCOP
      • INSTRUME
      • FILTER
      • RMFVERSN
      • CHANTYPE
      • DETCHANS
      • HDUCLASS
      • HDUCLAS1
      • HDUCLAS2
      • HDUCLAS3
      • HDUVERS1

    • A EBOUNDS table, containing the following
      • A 16-bit integer column labelled CHANNEL
