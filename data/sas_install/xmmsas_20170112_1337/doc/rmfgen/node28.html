EBOUNDS table creation

  • Write header information for EBOUNDS table

  • Get the channel ranges from the CAL and attempt to rebin them for the input spectrum. If this fails: For each spectrum channel, compute the values of the columns E_MIN and E_MAX from the PI channel range corresponding to that channel.

  • Write EBOUNDS data to file
