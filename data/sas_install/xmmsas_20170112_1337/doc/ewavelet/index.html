ewavelet

Abstract:

The task ewavelet detects sources in an image using the Mexican hat wavelet algorithm. The algorithm detects both point sources and extended sources, provided that the source extent is not too large ( 1/8th the image size). The final source list will contain source positions, count rates, plus an indication of the source extent.
