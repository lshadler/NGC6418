omthlcplot (omthlcplot-1.4) [xmmsas_20170112_1337-16.0.0]



Meta Index

omthlcplot

Abstract:

omthlcplot produces rate plots of the OM tracking star's intensities. The plots can be presented on a single page or on several pages, where the total counts can be binned up as required.
