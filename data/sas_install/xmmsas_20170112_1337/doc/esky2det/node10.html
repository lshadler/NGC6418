Input Files

  1. Required only if datastyle=`set': a source list in the form of a FITS table intab containing the following columns:

    • RA, in decimal degrees.

    • DEC, in decimal degrees.

    • If witherrorcol=`yes', either
      • columns RA_ERR and DEC_ERR, in decimal degrees, or:

      • RADEC_ERR, in arcseconds.

    All data types may be either REAL32 or REAL64.

  2. Required only if calinfostyle=`set': an XMM product file containing keywords INSTRUME, DATE-OBS, RA_PNT, DEC_PNT and PA_PNT.
