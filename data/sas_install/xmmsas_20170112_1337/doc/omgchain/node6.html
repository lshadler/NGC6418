Grism chain procedure

Omgchain employs seven SAS tasks (Fig.1).

Figure 1: OM Grism Mode Chain
The first four tasks are preparatory. For instance, if the OSW image is split into sub-windows, then these sub-windows are combined into a single file (see section 4.1). The necessary keywords, if missed, are inserted into the file headers. These tasks also implement the above-mentioned corrections to the image.

The other three tasks execute the source detection and spectral identification procedure, and produce corresponding output.
