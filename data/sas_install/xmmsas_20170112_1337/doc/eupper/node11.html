Statistical upper limit

By default this gives the 2-sigma upper limit on the background subtracted source count rate. If the number of counts is small, i.e. there are less than 80 counts in the source region then the Bayesian upper limit is returned (see Kraft, Burrows and Nousek, 1991, ApJ 374, 344).
