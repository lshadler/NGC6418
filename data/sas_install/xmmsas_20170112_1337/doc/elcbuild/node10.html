Comments

  • This task is primarily intended for PPS use. elcbuild was developed to fill the hole between epiclccorr and lcplot, for multiple time series. In this way several time series products can be plotted by the task lcplot.

  • All input files must be OGIP compliant, so that the output file will also be OGIP compliant and must contain the columns: RATE; ERROR; BACKV; BACKE; and FRACEXP. These files can be generated by epiclccorr.

  • The test is twofold: First it checks that the output file has been created, and then runs lcplot, produces a color postscript file, and finally checks on the validity of the postscript file produced.
