elcbuild

Abstract:

elcbuild takes a set of NTimeSeries time series which must contain the five columns: RATE; ERROR; BACKV; BACKE; and FRACEXP, as its input. The output file has the same extension and columns, but each column contains a two dimensional array (NtimeSeries:nRates), where nRates is the number of rates of the RATE extension. This output can then be plotted by lcplot.
