Output Files

EPIC-pn raw event list; two new columns, OFF_COR, containing the offset correction, and NOISE, which flags events as noise events are added to the event table. Column PHA is updated by task eprejectti (the original PHA value can be restored by adding the value in the OFF_COR column). If soft flare screening is enabled, an additional flag column, FLARE, will be created.
