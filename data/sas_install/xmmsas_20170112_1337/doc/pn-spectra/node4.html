regionyesintreg.txt 
the selection expression for the desired region for the generation of the model background spectrum. If no file with the input name exists, or if the file is empty, then the default is to model the data from the entire field of view. If a specific region is desired, the region expression must be in detector coordinates. For example, a file containing &&((DETX,DETY) IN circle(201,-219,3600)) would extract the central of the cluster Abell 1795. Note that the leading ``&&'' are required as the selection expression is added to other constraints.
