Using the CCF constituents

cifbuild can build a CIF by examining the CCF constituents found in a number of directories specified by the user. The list of directories (use the character : as a separator) should be specified through the environment variable SAS_CCFPATH. Alternatively, the same information can be specified on the command line (see the parameters withccfpath and ccfpath).
