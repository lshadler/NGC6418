Using a Master Index File

cifbuild can also be used to generate a CIF based on the contents of the Master Index File (MIF). The MIF contains the list of all CCF constituents released from the start of the mission until the date specified in the SUBDATE attribute. The generation of a CIF in this case requires that the user specify the MIF to be used with the parameters withmasterindexset and masterindexset.

The latest version of the MIF is available from the CCF area at the XMM-Newton Science Operation Centre.
