Input Files

odfingest requires access to a complete ODF. ODF components must comply with the ODF file naming convention.

The only ODF component that is required to be present is the ODF summary file. If this cannot be found odfingest terminates in error.

All instrument data files found in the ODF directory are analyzed to determine the structure of the observation in terms of active instrument and exposures. If any instrument science file is found, then all the corresponding housekeeping files must be present.
