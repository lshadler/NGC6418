Proposal Summary Record

The proposal summary record follows the very last configuration record and contains a summary of the proposal information asssociated with the observation. This is the information contained within the proposal database regarding the proposal submitter and the general observation details.

Only the first line of the proposal summary record will be present in the slew summary file.
