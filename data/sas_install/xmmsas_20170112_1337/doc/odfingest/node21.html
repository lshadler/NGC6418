Observation/ Slew Record

The observation/ slew record is the first record of the file and it will have the following structure.

Observation Summary File: Observation Record

Line Offset Type Description Note
No.
1 0 A11, 69X 'OBSERVATION' Note 1
      'SLEW '  
2 0 A10, 1X Observation Id. Note 2
2 11 A69 Comment Note 3
3 0 A4, 1X Orbit/ Revolution No. Note 4
3 5 A75 Comment Note 3
4 0 A20,1X Scheduled Start Time Note 5
4 21 A59 Comment Note 3
5 0 A20,1X Scheduled End Time Note 5
5 21 A59 Comment Note 3
         

  1. Identifies the record as an Observation or a Slew Record
  2. The syntax is ppppppooll (Section ) for an observation and TBD for a slew.
  3. All comments will have the syntax ' / text'
  4. The syntax is rrrr (Section )
  5. yyyy-mm-ddThh:mm:ss
