Event lists

Events are stored in a table in a data set. There is only one event list in a data set. The name of the table is 'events'.

Each row in a column can contain associated information. This has the form of a column in the table.

Work on the XMM event list definition is done by the SSC. The various columns in the event list would be described here, or a suitable reference given.
