evlistcomb

evlistcomb creates the combined event list from the set of intermediate event lists for one exposure. The EVENTS tables are merged and other tables, except for the PIXELS table, are copied whole, with the number of the originating CCD appended to the extension name to avoid name conflicts. The intermediate event lists are not required beyond this point.
