rgsbadpix

rgsbadpix adds the two node-specific BADPIXn tables to the intermediate event lists. These tables are a combination of known bad pixels from the static calibration data and bad pixels found by analyzing the PIXELS table. The task parameters allow either contribution to be suppressed.
