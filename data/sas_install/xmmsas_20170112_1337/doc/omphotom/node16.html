Recommendations

For FAST mode data, method 3 (PSF fit including background) is recommended. If the source has one or more close neighbours, then those sources should be included in the fit. Method 6 (PSF fit including background and a fit to the positions), may/may not give a slightly better fit, and should be used with caution.

For IMAGING mode data, if the source has one or more close neighbours, then method 3 (PSF fit to the source and its neighbours and including the background) is recommended if the source has one or more close neighbours. Method 6 (PSF fit including background and a fit to the positions), may give a slightly better fit, but should be used with caution.
