Input Files

  1. Required only if datastyle=`set': a source list in the form of a FITS table intab containing the following columns:

    • DETX.
    • DETY.

    if inputunit=`det' or

    • RAWX.
    • RAWY.
    • CCDNR.

    if inputunit=`raw'. All data types may be either REAL32 or REAL64.

  2. Required only if calinfostyle=`set': an XMM product file containing keywords INSTRUME, DATE-OBS, RA_PNT, DEC_PNT and PA_PNT. It also needs to contain the WCS keywords, CRPIX1 etc. to allow conversion to image pixels. A further set of keywords, REFXCRPX etc. allow translation to an X/Y tangential plane coordinate system.
