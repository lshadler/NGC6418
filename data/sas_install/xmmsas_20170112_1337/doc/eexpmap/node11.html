Future developments

  • DSS support: Currently only TIME range and GTI filters as well as CCDNR range filters are evaluated. DSS filtering of other event properties (spatial, pattern, energy ...) still needs to be implemented.
  • Coordinate transformations: Starting with version 4.6.1, eexpmap repeats the same errors as are in attcalc, and converts with different formulae for MOS and for PN, in order to match exactly the event coordinates calculated by emevents and epevents. These discrepancies and errors await resolving.
