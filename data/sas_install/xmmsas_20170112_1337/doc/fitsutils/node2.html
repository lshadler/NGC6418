Introduction

This library will consist of a series of Fortran 90 modules useful for handling FITS files. At the first delivery (Ver. 1.0), there is one Fortran library and one Perl library are available. Each module is described in a separate section below. Each module contains a separate file and will be compiled separately. The resulting object files will be combined into a single library file.
