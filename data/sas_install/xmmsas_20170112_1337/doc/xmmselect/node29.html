tableyesfilename table specifier
a specifier which must point to a table in a data set. It must be in either of the three forms setname, setname+tableid or setname[tableid] where setname must be the name of an existing data set and tableid an identifier of a table in that data set. If the first form, setname, is used, the table data are sought in the first block of the named data set. tableid can either be an identifier starting with a letter which must then be the name of a table in the specified data set or a simple number which is directly interpreted as a block sequence number (starting from 1).
