Meta Index

rgsfilter

Abstract:

This task filters the combined event list for RGS. The filtering is limited to time intervals and specific event flags, and is applied to the frames, events and exposure maps. This is not a generalized selection engine, but should be used for any filtering operations that impact on the exposure maps.
