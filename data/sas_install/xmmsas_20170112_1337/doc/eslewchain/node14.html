Comments

  • At points of inflection the interpolation of the attitude curve fails a bit. This produces errors of arcmin at the worst.
  • NB: The slew images have very few photons (typically about 100 in half a square degree). To source search these images, the task eboxdetect, should be used with the option obsmode=slew. Otherwise, a large number of spurious sources will be produced from these images.
