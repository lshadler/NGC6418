How to use

eslewchain runs on a processed event list in the current directory. A typical processing sequence would be:

  1. Download the slew datafile (SDF) into a clean directory and point to it with the environment variable SAS_ODF.
  2. Run ccfbuild and point to the CCF with SAS_CCF
  3. Run odfingest
  4. Run epproc in a working directory
  5. Set the SAS_ATTITUDE environment variable to "RAF".
  6. Run eslewchain in the directory containing the event file produced by epproc.
