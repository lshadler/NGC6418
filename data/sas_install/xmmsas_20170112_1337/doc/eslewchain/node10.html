Input Files

The input files are those contained in a directory after odfingest and an EPIC proc or chain have been run on a slew data file (SDF). In particular the directory must contain:

  1. An EPIC event file (only one should be present in the directory).
  2. A Raw Attitude File, "rev"_"obsid"_SCX00000RAS.ASC
  3. A summary File, "rev"_"obsid"_SCX00000SUM.SAS
