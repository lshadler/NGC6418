Output Files

The output is located in different directories, prep_mosaic_xxx, with xxx running from 001 to the maximum of pointings allowed within one observation (). In each directory, an event file per EPIC instrument used is created containing the events corresponding to the given pointing in the sequence. On top of that, several links to the input event files as well as to SAS summary file and attitude information file are created.
