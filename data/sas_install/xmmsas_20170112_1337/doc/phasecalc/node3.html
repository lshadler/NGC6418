Description

Given an ephemeris and a list of event tables, phasecalc calculates the phase of each event in each of the tables. The calculated values are stored in a column named PHASE. If the column exists it is overwritten.

phasecalc relies on the event table having a TIME column and the two attributes MJDREF and TIMEZERO.

  • MJDREF is the Modified Julian Day from which times are calculated.

  • TIMEZERO is the clock correction at the start of the observation in seconds.

  • phasecalc also uses the TIMESYS keyword to determine the Time System (UTC, TT, TAI) used in event table. If the TIMESYS keyword is not UTC, the Reference Epoch time of the ephemeris is converted to the appropriate Time System. In the absence of this keyword TT (XMM Time) is assumed.
