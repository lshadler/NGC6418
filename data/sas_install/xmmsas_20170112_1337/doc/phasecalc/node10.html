Comments

phasecalc is meant to be used in conjunction with evselect or xmmselect to perform second order filtering.

phasecalc is not a general tool, in that it has a very specific knowledge of how to calculate the absoulte time of an event based on the XMM-Newton conventions. On the other hand, this affords a paramter interface focussed on the astronomical data. For a more generic tool to perform arbitrary math on binary tables see tabcalc ( ).
