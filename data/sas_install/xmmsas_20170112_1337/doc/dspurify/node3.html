Description

Purify a dataset.

This task removes non-compliant dal data from a dataset.

e.g. X columns (the DAL is unable to read datasets containing X columns) will be removed from a dataset.

An optional dataset can be specified to which the 'clean' dataset will be written, otherwise the original datase is modified.
