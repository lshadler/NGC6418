Description

tmcheck will open the required raw TM datafiles, and for those DU (NCTRS frames) with the time stamp within the entered time range,
- Datafiles info will be displayed (verbosity 4, sparse)
- DU and Frames statistics will be displayed (verbosity 5, verbose)
- Packet statistics will be displayed (verbosity 6, noisy).
- Packet dumps will be generated (verbosity 7).

Note that current implementation will only use packets belonging to VC0.
