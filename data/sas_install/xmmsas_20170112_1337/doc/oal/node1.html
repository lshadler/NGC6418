Introduction

The ODF Access Layer library provides ODF-related utility functions and gives access to a limited set of non-science ODF data sets. The purpose is to provide a central repository for ODF-access functionality which is likely to be needed by more than one SAS task. Please note: The OAL is not intended to fully insulate the SAS tasks from the structure and contents of the data sets in the ODF. This is especially true for the science data sets which must be directly read by the task via the XMM Data Access Layer package dal.
