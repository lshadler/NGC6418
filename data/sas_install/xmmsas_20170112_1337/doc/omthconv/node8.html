Algorithm 

   subroutine omthconv

   Read in parameters.

   Get a handle on the ODF OM Tracking History Data Auxiliary File.

   Determine datamode from keyword in primary header.

   Get a handle on the tracking history data binary table extension.

   Get start and end of exposure time stamps (OBT) from the ODF Non-Periodic Housekeeping File

   Convert Onboard start/end Timestamps (OBT) to event time tags (elapsed seconds since mission 
   reference time 1998-01-01T00:00:00) using OAL_obtToTimeTag call

   Create and get handle on output event list file.

   Write required keywords from Tracking History primary header to output file primary header.

   Create and get a handle on RATES extension.

   Write attributes

   Write RATEn columns

   Free up memory and pointers.

   Release handles on data.

   end subroutine  omthconv
