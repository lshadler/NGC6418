The task eimsim

It is not expected that the user will run this task directly: instead, the user should run eimsimbatch, which calls eimsim times, where is a user-specifiable number. All necessary eimsim parameters are also parameters of eimsimbatch, which passes them straight through to the subtask. eimsimbatch just contains a loop for performing the calls, plus some machinery for cleaning up, and for `nice' stopping. Thus it makes sense to place most of the detailed description of the simulation within the present document rather than in the documentation for eimsimbatch.

Task eimsim does 9 things in sequence, described in the following subsubsections:


Subsections
  • Make list of simulated sources
  • Flag those sources outside the region of detectability
  • Make a counts/pixel/sec image
  • Make a counts/pixel expectation-value image
  • Make a random, Poisson-distributed events image
  • Perform source detection
  • Transform the flux coordinate
  • Add various bits and bobs
  • Attempt to match detected and simulated sources
