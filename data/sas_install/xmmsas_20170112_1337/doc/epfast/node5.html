Introduction

The epfast task is designed to correct for a CTI effect which has been seen in EPIC-pn fast mode (timing, burst mode) data. Firstly, it finds the count rate of electrons which have been recorded at a given time in a given column of the CCD. It then applies a formula to this rate to calculate a correction value for the CTI.

The number of electrons is derived from the total PI charge deposited on the column at that time.
