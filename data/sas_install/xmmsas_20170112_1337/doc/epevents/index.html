epevents

Abstract:

This task converts an EPIC PN raw events file (produced by epframes and badpix) into a calibrated event list with flagged trailing events and corrected for gain variations of the preamplifier channels and charge transfer inefficiency (CTI). It performs a pattern analysis to identify and recombine split events and converts the raw unlinearized detector coordinates into corresponding physical locations.
