MIPs:

If the whole pattern contains at least one pixel with an amplitude above a certain threshold (e.g. 3000 adu), this pattern is interpreted as having been caused by a minimum ionizing particle (MIP), and all related pattern parameters are NULL and PHA_CTI and PI are NULL, to indicate that these events should be ignored.
