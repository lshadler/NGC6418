Input Files

  1. FITS image, in any of the data types output by evselect. The image must be the primary image of the FITS file.

  2. A FITS source list. This table must have columns called RA and DEC (in decimal degrees). If radiusstyle=`psf', it must also have columns called SCTS and BG_MAP.
