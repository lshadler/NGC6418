Setting the names of output files individually

It is possible to set the names of the output files separately without using the filestem keyword by:

especget srcspecset=agrb.pi bckspecset=cleansky.pi srcarfset=agrb_arf.ds srcrmfset=agrb_rmf.ds srcexp="((X,Y) IN circle(25572,-24635,1200))"
backexp="((X,Y) IN circle(23000,-28000,1340))"
table=P0129360201PNS020PIEVLI0000.FIT
