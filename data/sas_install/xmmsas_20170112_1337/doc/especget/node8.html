Meta Index / Home Page / Description / Examples

A point source

especget filestem=myagn srcexp="((DETX,DETY) IN circle(572,-635,800))"
backexp="((DETX,DETY) IN annulus(572,-635,1600,3200))"
table=P0129360201M1S004MIEVLI0000.FIT

This command line produces the files myagn_src.ds, myagn_bgd.ds, myagn_src.arf, myagn_src.rmf.

The command will take quite a while to run because of the RMF generation stage . A PN observation processed with this command will take roughly twice as long as a MOS observation.
