Using a pre-prepared matrix

especget filestem=myagn srcexp="((X,Y) IN circle(25572,24635,1200))"
backexp="((X,Y) IN circle(23000,28000,1340))"
table=P0129360201PNS020PIEVLI0000.FIT withrmfset=yes
rmfset=epn_ef20_sdY9.rmf

This command tells the task that the canned matrix epn_ef20_sdY9.rmf will be used for spectral fitting. The task should finish within a few minutes. NB: If rmfset is not defined then especget will set the name to that of the relevant canned matrix.
