Fast chain tasks

The Fast chain works in the following way:

  • When started, omfchain creates indexes of the Optical Monitor Fast mode data set files presented in the input directory.

  • If flat field files (FFX) are not found, omfchain creates an empty flat field file

  • If periodic or non-periodic housekeeping files are not found in the input directory, the chain is stopped.

  • If tracking history files (THX) are not found for the given exposure, the chain creates a dummy tracking history file.

  • For each observation, as well as for each exposure and scientific window, omfchain produces intermediate and pipeline output files according to the following scheme.
