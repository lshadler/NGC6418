Description

This package contains a PERL script which takes a set of fast mode OM ODF files in a single directory and processes them in a pipeline to produce OM fast mode products. Given a directory with a set of ODF files that conform to the filename specification given in the ODF ICD the PERL script will process the fast mode event list files. The chain produces images of the sources found in OSWs, determines regions for the sources and background, extracts events related to the source and background, and constructs corresponding lightcurves.
