Fast data types

Fast mode data is taken in a single format, arriving as an event list. In a given exposure up to two Fast mode windows may be defined, each giving rise to a separate event list. One should be aware that the Fast mode data are usually accompanied by Image-mode windows from the same exposure which could be used for more accurate determination of the background level (especially when the target source is bright).
