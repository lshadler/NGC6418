Mode dependent livetime factor FC0

The first livetime factor is mode dependent and constant for each exposure. It corresponds to the ratio of integration time to total time. The frame read-out dead-time is calculated in epframes already and appears (implicitly) in the TIMEDEL entries in the EXPOSURE extension which are smaller than the frame interval by a factor FC0. This factor is also written by epframes into the keyword AX_FC0.
