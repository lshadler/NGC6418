Input Files

  • srclist An OM source list generated by the OM processing chain. The code expects this file to have a table SRCLIST which contains the columns RA_CORR, DEC_CORR, filtername_CORR_RATE and filtername_CORR_RATE_ERR.

    Typical files to use would be the COMBINED source list with a name like PooooooooooOMCOMBOBSMLI0000.FIT or the merged source list with name PooooooooooOMCOMBOBSMER0000.FIT.

    where oooooooooo specifies the 10 digit observation ID.
