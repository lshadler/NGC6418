Before you start...

Before you start the odfbrowser, you have to make sure that at least the environment variable SAS_ODF is pointing to an ODF summary file created by odfingest.

If you plan to run one or more procs on a selection of exposures, you will also have to point the environment variable SAS_CCF to a CCF file created by cifbuild.

For example (sh): 

   > export SAS_ODF=/data/0174_0080940401_SCX00000SUM.SAS
   > export SAS_CCF=/data/ccf.cif
