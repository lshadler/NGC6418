Using exposures

You can simply (de-)select exposures by clicking on them.

Selecting one or more exposures activates one or more configuration buttons in the tool bar. These buttons allow you to adapt the procs to your needs. At the same time, a start button is enabled. By pressing this button, odfbrowser will launch all procs that are needed to process these exposures sequentially.
