Meta Index / Home Page


Algorithm

The program is written in c++ and performs two different tasks:

  1. Populates the QUALITY image extension within individual OM images using the source-list data produced by omdetect.

  2. To set the quality flags of sources in an OM source-list produced by omdetect, using the information stored in the quality image of the image file.


Subsections
  • Setting quality-image pixels (Program mode 1)
  • Setting of bit 1 (Read-out streak)
  • Setting of bit 2 (Smoke-rings)
  • Setting of bit 3 (Diffraction-spikes)
  • Setting of bit 4 (Modula-8 pattern)
  • Setting of bit 5 (Within central-enhancement region)
  • Setting of bit 7 (Near image edge/ corner of OM detector)
  • Setting of bit 9 (Isolated exceptionally bright pixels)
  • Setting source-quality flags
