Input Files

  1. PPS product (from task evselect): FITS images (one per instrument per energy band if program is run in imaging mode - default)
  2. PPS product (from task evselect): EPIC IMAGING-mode event lists (one per instrument if program is run in Photon mode - not yet implemented)
  3. PPS product (from task eboxdetect run in map-detect mode): EPIC eboxdetect source list
  4. PPS product (from task eexpmap, optional): EPIC exposure images (one per instrument)
  5. PPS product (from task esplinemap, optional): Spline background images (one per instrument per energy band)
  6. PPS product (from task emask, optional): Detection masks (one per instrument)
