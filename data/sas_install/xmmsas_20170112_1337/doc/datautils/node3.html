Description

This is a collection of C++ classes used to augment data manipulation functionality present in the DAL.

This library includes classes to

  • Provide automatic type conversions (ConvertableData)
  • Support pixel arrays (images) and pixel lists

  • Read in and perform simple (linear) mappings of WCS information stored in datasets.

The main client of this package is arfgen.
