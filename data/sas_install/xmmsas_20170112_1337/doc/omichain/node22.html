Rotated sky images (FITS files)

Omatt produces a rotated sky-image (RA-DEC). The SAS task slconv can be used to produce a ds9 region file from the source-list file produced by omatt, and then the sky image can be displayed using ds9 and the sources displayed using the region file. Ommosaic produces a sky image, for a given filter, from all the sky-images produced by omatt for that filter.

  1. Sky image produced by omatt - eg P0123920101OMS006SIMAGE1000.FIT
  2. Sky image produced by ommosiac for a given filter - eg P0123920101OMS000RSIMAGV.FIT
