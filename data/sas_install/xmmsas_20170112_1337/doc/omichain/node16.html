Pipeline Processing System (PPS) Product Files

Nearly all the product files are in FITS file format, with the extension FIT, and all begin with the letter P. In the following examples of file names, XX represents a code for an OM filter- XX=U for U filter, B for B filter, V for V filter, WL for WHITE (unfiltered), W1 for UVW1, M2 for UVM2 and W2 for UVW2. In what follows, OSW stands for OM Science Window.
