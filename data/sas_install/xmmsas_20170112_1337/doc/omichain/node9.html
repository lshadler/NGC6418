Production of the mosaiced sky-image

When all the exposures for a given filter have been processed, ommosaic is used to combine the low-resolution sky-coordinate images into a single image. ommosaic will only run if there is at least one sky image for the filter.) Before using an image the RA and DEC the astrometric corrections computed by omatt are applied to the reference pixel coordinates.
