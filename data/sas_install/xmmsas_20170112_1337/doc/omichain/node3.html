Description

This package contains a PERL script which takes a set of imaging mode OM ODF files in a single directory and processes them in a pipeline to produce OM imaging mode pipeline products. Given a directory with a set of ODF files that conform to the filename specification given in the ODF ICD the PERL script will process the image files and produce output files conforming to the ICD specifications.
