Unrotated image-files (fits)

Omprep produces the first intermediate image file, by converting the integer image in a raw image file into a real image and adding some keywords to the fits header. Omcosflag then adds a Quality image to this same file. Omflatfield then flatfields this image by dividing it by the master flatfield produced by omflatgen.

  1. Real image file produced by omprep - eg I0123920101OMS006IMAGE10000.FIT
  2. Image file modified by omcosflag - eg I0123920101OMS006IMAGE10000.FIT
  3. Image file produced by omflatfield - eg I0123920101OMS006IMAGE20000.FIT
  4. Image file produced by omcomb - eg I0135720601OMS01700E2I.FIT

Please note that the latter image is only produced if there are 3 or 4 engineering-2 exposures.
