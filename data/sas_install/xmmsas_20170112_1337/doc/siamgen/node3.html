Description

siamgen generates the SIAM matrices as specified in the Interface Control Document[2] for the six instruments EMOS1, EMOS2, EPN, RGS1, RGS2, OM from the contents of the CCF Boresight constituent. For each instrument the cal is inquired for the corresponding matrix on which siamgen performs a basic ICD-compliance check (see below). If that checks succeeds the matrix elements are written to stdout preceded by a header record containing the instrument ID and time when the boresight calibration for this instrument was performed.

In order to provide the SIAM matrices to FDS according to the Interface Control Document[2] the output of siamgen should be redirected into a file and wrapped into a suitable XFTS wrapper[1].

siamgen has no mandatory parameters. For the list of optional parameters see below.
