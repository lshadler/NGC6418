Routines which return information about DSS components

Each DSS component contains several filters. Each filter is associated with the name of the event-list column (called in DSS-speak the 'axis' name) which was filtered (for some filter types, eg REGIONFILTERT, two names are supplied), plus a specification for a set of values of that (or those) column(s).
