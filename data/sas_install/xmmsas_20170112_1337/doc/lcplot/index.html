Meta Index

lcplot

Abstract:

lcplot produces plots of the net source and background time series for a single time series FITS file or for several time series FITS files produced by elcbuild for EPIC and omlcbuild for the OM. Furthermore it performs either the Chi Squared probability of constancy test and/or the Kolmogorov-Smirnov probability of constancy test on the light curve to determine whether the source is variable.
