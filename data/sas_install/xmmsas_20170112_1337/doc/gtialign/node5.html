`Generic' mode

This is the new mode which was introduced in version 2.9. To access this mode you should set style=`generic'. In this mode the task takes two inputs, supplied to parameters ingtitable and tstable, and generates a single GTI table as output, which is written to the dataset+table specified via parameter outgtitable. Otherwise the functioning is similar to the `pipeline' mode.

An example command: 

gtialign style=generic ingtitable=my_gtis.ds:STDGTI01 tstable=lccorr_out.ds:RATE
outgtitable=some_file.fits:NEW_GTI
