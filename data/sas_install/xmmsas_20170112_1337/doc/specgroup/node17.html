Default behaviour

The command:



will have the following behaviour:

  • Channels defined as bad for that instrument in the corresponding XMM_SPECQUAL CCF will be set BAD.
  • All keywords, ANCRFILE, BACKFILE, RESPFILE, will be set to "NONE"
  • The data will be grouped such that each input channel is in a group of its own
