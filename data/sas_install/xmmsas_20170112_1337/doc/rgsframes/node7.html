Output Files

The only output is a single dataset containing all information needed to process a CCD. The structure of this intermediate event list is described in the rgsproc.

It also sets the following attributes:

  • TIMESYS = TT
  • MJDREF = 50814
  • TIMEZERO = 0
  • TIMEUNIT = s
  • CLOCKAPP = T
  • TIMEREF = LOCAL
  • TASSIGN = SATELLITE
  • SUBMODE
  • CCDMODE
  • CCDMODE
  • SINGLERO
