Detailed layout of GUI

When multixmmselect is invoked it comes up with a main window as illustrated in the following figure:





Figure: Layout of elements in main multixmmselect window

The window consists of the same main areas as in the xmmselect task, but in this case, several features, such as: PATTERN, FLAG, etc... selections have been disabled to limit the usage of multixmmselect only to OGIP spectral product generation.
