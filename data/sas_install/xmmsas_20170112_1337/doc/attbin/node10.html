Input Files

  1. (if attsource=`odf' and timesource=`eventlist'), a dataset containing a binary table extension EVENTS, which contains a 4-byte-real-valued column TIME.

  2. (if attsource=`odf' and timesource=`eventlist'), a file (usually expected to be the output file of atthkgen) which contains in the primary HDU keywords as follows:

    • if odfattsource=`ahf':
      • if withmedianpnt=`yes': MOMRA MOMDEC MOMPA
      • if withmedianpnt=`no': AOMRA AOMDEC AOMPA
    • if odfattsource=`om':
      • if withmedianpnt=`yes': MAHFRA MAHFDEC MAHFPA
      • if withmedianpnt=`no': AAHFRA AAHFDEC AAHFPA

  3. (if attsource=`atthk'), a file (usually expected to be the output file of atthkgen) which contains in the primary HDU keywords as described in the previous item, plus an extension ATTHK, which contains the following columns:

    • TIME
    • (if atthkattsource=`ahf') AHFRA, AHFDEC and AHFPA
    • (if atthkattsource=`om') OMRA, OMDEC and OMPA

    All these columns should be 4-byte real.
