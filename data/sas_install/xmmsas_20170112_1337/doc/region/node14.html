Output Files

The task in general writes two files, one specified by parameter regionset for the source region(s) and the other, specified by parameter bkgregionset, for the background region(s). However the exact details depend on the value of the operationstyle parameter as follows:

  • operationstyle=`single': task writes both source and background files; each contains a single extension called REGION in both cases.

  • operationstyle=`batch': task writes both source and background files; each file contains one extension per source, these being named REG_nnnn, where the nnnn are consecutive integers starting at 1.

  • operationstyle=`global': task writes background file only; extension name is REGION; this contains the global background region, which is the entire projection plane minus source cutouts.
