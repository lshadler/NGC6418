Description

This library contains 3 modules for F90 :
  • emutils_mod (Sect 2.1)
  • edusoft_mod (Sect 2.2)
  • badpixutils_mod (Sect 2.3)
They are described in the following subsections.

It also contains the test directory emodf for the EPIC MOS routines, and the utilities compare_columns and compare_realcols used in test harnesses.
