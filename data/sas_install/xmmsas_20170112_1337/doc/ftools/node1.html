Running FTOOLS

The SAS GUI provides limited support for running FTOOLS, in additional to native SAS tasks, although they are naturally not as well integrated.

This package contains SAS parameter files and layout files for some common FTOOLS. These customise the parameter dialogs to make best use of the SAS GUI.

When the FTOOL is started from the GUI, it is passed a command line containing a complete set of parameters as defined in the custom parameter file. The parameter type in the custom SAS parameter file can, for example, be an SAS `file' or `set', when the FTOOL parameter file specifies a `string' parameter. The FTOOL interprets the parameters using its own parameter files.

There are some limitations on the use of FTOOLS (see below).
