The task eboxdetect takes as input images of the same part of the sky in separate energy bands, and (in `map' mode) images of the estimated background counts per pixel. The task applies a sliding box to these stacked images. At each pixel, then, the task has available values of the actual number of events found within the box and values of the expected number of background events within the same box.
