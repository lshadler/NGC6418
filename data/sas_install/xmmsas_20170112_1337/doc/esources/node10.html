Output Files

  1. A FITS dataset which contains a source list in the form of a binary table extension named SRCLIST. This table contains the following columns:
    • SRC_NUM (INTEGER32)
    • RA (REAL64)
    • DEC (REAL64)
    • RA_CORR (REAL64)
    • DEC_CORR (REAL64)
    • TBIN_WIDTH (REAL64)
    • SRC_NUM_HEX (STRING)
    • PROCESS (INTEGER8)
    • EP_CTS (REAL32)
    • PN_CTS (REAL32)
    • M1_CTS (REAL32)
    • M2_CTS (REAL32)
    • ML_ID_SRC (INTEGER32)

  2. The SPECTRA and TSERIES columns of the input source table (srctab) are written to if setflags=`yes'.
