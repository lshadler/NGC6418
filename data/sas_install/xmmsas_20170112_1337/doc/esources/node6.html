Start and stop times

Again to facilitate comparison of time series, the same start and stop times are used for all EPIC source time series from a given observation. These times are obtained from TSTART and TSTOP keywords in the EVENTS table headers of the PN, M1 and M2 event lists. The start time is taken as the minimum of the TSTART values and the stop time as the maximum of the TSTOP values.
