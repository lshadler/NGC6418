Input Files

  1. an EPIC events list from the pipeline (e*proc or e*chain tasks)
  2. an EPIC filter-wheel-closed events list, available at
    http://xmm2.esac.esa.int/external/xmm_sw_cal/background/filter_closed/index.shtml
  3. an EPIC out-of-time events list (only for EPIC-pn, epproc or epchain tasks)
  4. optionally, a good-time-interval file (e.g. from tabgtigen)
  5. an attitude file (only needed if exposure maps will be created or the coordinate frame is changed)
