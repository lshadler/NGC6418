Execute a task

At the bottom of the task dialog you will always find three buttons:
  • Run
    Starts the tasks with the selected parameters. Any changes you have made to the parameters of this task, will be kept in memory.
  • Cancel
    This button will close this window. Any changes to the parameters are ignored.
  • Save
    This button will store your current parameters value in the $HOME/.sas.d/config directory.
  • Defaults
    Reverts the parameters to the values they had, when you opened this window.
