Description

This task plots the first and second order spectra (Counts/channel versus BETA CHANNEL) for an RGS source, overlaying the source dependent energy and wavelength scales if the data are provided in radians or counts/channel against wavelength if the data are provided in wavelength space. The data can be represented in original format, or rebinned, to contain a minimum number of counts (specified by the user) per bin. In this case, the counts/channel for each bin are plotted verses the centre of that bin, as a histogram. The error values are plotted in gray and are always the square root of the counts. Details of the observation and source are also plotted. Any valid PGPLOT graphics device may be specified for the output.
